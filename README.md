# Nudge

A macOS menu bar utility for window management and quick text editing.

## Features

- **Window management** — move, resize, and snap windows of any app from the keyboard or with modifier-drag.
- **InstantEditor** — a pop-up scratch editor for composing or tweaking text before pasting it into the app you were just in.

## Install

1. Download the latest `nudge-v*.dmg` from the [Releases page](https://github.com/saryou/nudge-releases/releases/latest).
2. Open the DMG and drag `nudge.app` into `/Applications`.
3. Launch Nudge. On first run, grant **Accessibility** permission when prompted (System Settings → Privacy & Security → Accessibility).

The app updates itself automatically via Sparkle.

## Requirements

- macOS 14.0 (Sonoma) or later
- Apple Silicon and Intel both supported
